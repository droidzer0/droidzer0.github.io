            How to use
            
    
                
  1. Paste or upload your JSON into the editor (you can also drag and drop a .json file).
    2. 
                
  2. Select the desired functionality. For Field/BOM, you can enter multiple values (comma or newline separated), or enable regex matching.
    3. 
                
  3. Click "Parse Data". Use search boxes and filters to refine results. Click column headers to sort; drag column edges to resize.
    4. 
                
  4. For Elapsed Time, adjust Top N and the minimum threshold. Use the Download button to export current view.
